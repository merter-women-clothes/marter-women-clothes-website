# Marter Women Clothes

متجر عربي متجاوب لبيع الملابس النسائية بالجملة في العراق، مع لوحة إدارة للمنتجات ورفع الصور.

تتضمن نسخة النشر الحالية 16 منتجاً و119 صورة مرتبطة بها. الصور الافتتاحية مجمعة داخل `server/seed-uploads.zip` لتسهيل رفع المشروع، وتُنسخ تلقائياً عند أول تشغيل على الخادم. بعد ذلك تُحفظ التعديلات اللاحقة على القرص الدائم.

## التشغيل المحلي

```bash
npm install
npm run dev
```

- المتجر: `http://localhost:5173`
- لوحة الإدارة: `http://localhost:5173/admin`
- كلمة المرور المحلية الافتراضية: `master2026`

## الإعداد قبل النشر

1. رابط واتساب واسم مسؤول الطلبات محفوظان في `src/config.js` ويمكن تحديثهما من هناك.
2. غيّر كلمة مرور الإدارة عبر متغير البيئة `ADMIN_PASSWORD`.
3. حساب تيليجرام مضبوط على `master_women_clothes` ويمكن تغييره من `src/config.js`.

## النشر

المشروع جاهز للنشر باستخدام `Dockerfile`. كما يتضمن `render.yaml` للنشر على Render مع قرص دائم يحفظ المنتجات والصور المرفوعة من لوحة الإدارة.

```bash
npm run build
npm start
```

في الإنتاج يجب ضبط:

```env
ADMIN_PASSWORD=كلمة-مرور-قوية
DATA_DIR=/data
PORT=3000
```
